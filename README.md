# Volalert


## Install

Clone repository and run:

```sh
$ npm install
```

Deploy a copy to Heroku :

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/dougkna/volalert)

## Requirements

node 5+
